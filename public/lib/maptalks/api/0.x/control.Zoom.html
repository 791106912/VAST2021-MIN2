Zoom

A zoom control with buttons to zoomin/zoomout and a slider indicator for the zoom level.

new control.Zoom() [source] 

var zoomControl = new Zoom({
    position : 'top-left',
    slider : true,
    zoomLevel : false
}).addTo(map);
